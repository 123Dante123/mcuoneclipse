This folder contains Eclipse (Kepler) projects for various boards.
See this list of tutorials:
http://mcuoneclipse.com/2013/07/20/dyi-free-toolchain-for-kinetis-part-1-gnu-arm-build-tools/

Please note that the examples in this folder are rather old and have been created in the pre-Kinetis Design Studio area and are not maintained.
Please consider using Kinetis Design Studio and the examples provided in this repository for it.
