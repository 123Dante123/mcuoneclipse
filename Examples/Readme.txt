This folder contains examples using the Processor Expert components.
The projects are organized by IDE's and boards.

- Eclipse: Stock Eclipse Kepler/Luna/Mars with GNU ARM Eclipse plugins
- IAR: Projects with IAR Embedded Workbench
- KDS: Projects with Kinetis Design Studio V3.0.0
- Keil: Projects with Keil MDK
- RNet: Early version of a RNet (SMAC and nRF24L01+ stack), this has been moved to a Processor Expert component later
- FreeRTOS_Ports: various ports of FreeRTOS
