----------
Project for KDS 3.2.0 using Processor Expert to program the FLASH.
See https://mcuoneclipse.com/2014/05/31/configuration-data-using-the-internal-flash-instead-of-an-external-eeprom/


original FLASH area:
- name: m_text
- address: 0x410
- size: 0x1FBF0

Note that the clock settings in the IntFLASH component had to be set to 12 MHz.
