Example project for Adafruit 1.5" OLED breakout board (https://www.adafruit.com/product/1431)
with Kinetis Design Studio V3.2.0.

It includes the 'Electronic Animated Eyes' (thanks to Adafruit and Phil Burgess / Paint Your Dragon
for the incredible useful and amazing work!)


run the Python program to convert images, e.g.
(copy files to the same directory as the script)
c:\Python27\python.exe tablegen.py sclera.png iris.png upper.png lower.png 80 > test.h
