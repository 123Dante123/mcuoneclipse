----------
Example project for Adafruit 1.5" OLED breakout board (https://www.adafruit.com/product/1431)
with Kinetis Design Studio V3.2.0.

The project is using LittlevGL GUI library.

 
Remarks:
- non-LDD SPI does not work
- display orientation changes between buffered and non-buffered setting
- LE and BE issue
- Update partial does not correctly support area
