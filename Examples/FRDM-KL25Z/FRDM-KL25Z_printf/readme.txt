This project is demonstrating using printf() on the FRDM-KL25Z board, without Processor Expert.

COM port settings:
- UART0, PTA1 and PTA2: this is connected to OpenSDA USB CDC
- 38400 baud

For more information, see:
http://mcuoneclipse.com/2013/04/17/printf-with-the-frdm-kl25z-board-and-without-processor-expert/
