This project uses the ESP8266 WiFi module with the FRDM-K64F board, using MCUXpresso IDE.

See:
http://mcuoneclipse.com/2014/11/30/tutorial-web-server-with-the-esp8266-wifi-module/
http://mcuoneclipse.com/2014/10/15/cheap-and-simple-wifi-with-esp8266-for-the-frdm-board/
http://www.electrodragon.com/w/Wi07c
https://orxor.wordpress.com/2015/01/30/esp8266-intro/

The project has been ported from the original KDS project using the approach described in
https://mcuoneclipse.com/2018/07/27/porting-processor-expert-projects-to-mcuxpresso-ide/
