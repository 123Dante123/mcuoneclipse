Example project for Adafruit 1.5" OLED breakout board (https://www.adafruit.com/product/1431)
with Kinetis Design Studio V3.2.0.

It includes the 'Electronic Animated Eyes' (thanks to Adafruit and Phil Burgess / Paint Your Dragon
for the incredible useful and amazing work!)

Note: this project has been ported from Kinetis Design Studio V3.2.0 to MCUXpresso IDE 10.2.
Original project: https://github.com/ErichStyger/mcuoneclipse/tree/master/Examples/KDS/FRDM-K64F120M/FRDM-K64F_Adafruit_SSD1351

The porting process is described in:
https://mcuoneclipse.com/2018/07/27/porting-processor-expert-projects-to-mcuxpresso-ide/

Links:
- https://github.com/adafruit/Adafruit-1.5inch-Color-OLED-PCB
