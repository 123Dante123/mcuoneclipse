This is a room monitor project for the tinyK22 board which uses the same MCU as the FRDM-K22 board.

The original project is a Processor Expert project (https://github.com/ErichStyger/mcuoneclipse/tree/master/Examples/KDS/tinyK22/tinyK22_SGP30)

For porting such a project, see https://mcuoneclipse.com/2018/07/27/porting-processor-expert-projects-to-mcuxpresso-ide/
