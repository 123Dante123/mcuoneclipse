readme.txt
----------
Project for the FRDM-K22F with FatFS and FreeRTOS, see
https://mcuoneclipse.com/2020/05/20/fatfs-minini-shell-and-freertos-for-the-nxp-k22fn512/

Configuration of the project can be changed using the #defines in platform.h.
The project supports both FatFS with SD card and USB MSD (host mode).
